Infografía de JavaScript

Año de Desarrollo: JavaScript fue creado por Brendan Eich en 1995.

Versiones: ECMAScript es el estándar en el cual se basa JavaScript. Algunas versiones clave de ECMAScript incluyen ES5, ES6, ES7, etc.

Consideraciones Generales del Lenguaje: JavaScript es fácil de aprender y se utiliza para agregar interactividad a las páginas web.

Potencia del Lenguaje: JavaScript se ejecuta en el navegador del usuario y utiliza una sintaxis sencilla similar a otros lenguajes de programación.
