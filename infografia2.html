Infografía sobre CSS

Menú Hamburguesa de CSS

El menú hamburguesa es un icono de tres líneas horizontales que se utiliza en el diseño web para representar un menú desplegable en dispositivos móviles o pantallas de tamaño reducido. El icono se asemeja a una hamburguesa, de ahí su nombre.

Al hacer clic en el icono de la hamburguesa, el menú desplegable se expande, mostrando las opciones de acceso que pueden estar ocultas en una barra de navegación en la parte superior de la página en pantallas más grandes. Esto permite a los usuarios acceder a las diferentes secciones del sitio web de una manera más fácil y organizada en pantallas de menor tamaño.

Modelo de Caja de CSS

El modelo de caja de CSS describe cómo se representan visualmente los elementos HTML en una página web. Cada elemento HTML se considera una "caja" que tiene contenido, relleno, bordes y margen. El modelo de caja de CSS especifica cómo se calculan las dimensiones y posiciones de estos elementos, lo que permite al diseñador controlar el diseño y la disposición de los elementos en la página.

Etiqueta DIV de CSS

La etiqueta "div" de HTML es un elemento de bloque que se utiliza comúnmente como contenedor genérico para agrupar otros elementos y aplicar estilos a través de CSS. Por sí misma, una etiqueta "div" no tiene ningún significado semántico, pero se utiliza para organizar y estructurar el contenido de una página web de manera que sea más fácil de manejar y estilizar con CSS.

Diferencia entre Clase y Pseudoclase CSS

Clase CSS: Una clase en CSS es un identificador que se utiliza para aplicar un conjunto específico de estilos a uno o más elementos HTML. Se define con un nombre precedido por un punto (.) en el archivo de estilo CSS y luego se aplica a los elementos HTML utilizando el atributo class. Por ejemplo, en CSS podrías tener .mi-clase { color: blue; } y en HTML "div class="mi-clase">Contenido div". Esto haría que el texto dentro del "div" se mostrara en azul.

Pseudoclase CSS: Una pseudoclase en CSS es una palabra clave que se añade a un selector para especificar un estado especial del elemento seleccionado. Las pseudoclases permiten aplicar estilos basados en interacciones del usuario o estados del elemento, como :hover para cuando el usuario pasa el ratón sobre un elemento, o :first-child para seleccionar el primer hijo de un elemento. Por ejemplo, a:hover { color: red; } cambiaría el color del texto de un enlace cuando el usuario pase el ratón sobre él. Las pseudoclases se agregan al final de un selector, después de un dos puntos (:).
